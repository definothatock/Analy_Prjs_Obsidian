
| *TERM*          | *DEFINITION*                                                                                                                                              |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Game            | The instantiated and running activity of play as a whole.<br>- Running session, the “ongoing game”                                                        |
| Project         | The infrastructure (game-as-product) being analyzed.<br>- Artifact, Ecosystem, Combined Systems, etc                                                      |
| Component       | A discrete design element.<br>- unit, card, weapon, tile, etc                                                                                             |
| Mechanic        | A rule-bound interaction framework available in play.<br>- Some of the common "Genre" tags like 'Platformer'                                              |
| System          | A set of interacting mechanics/rules that produces Outcomes or Behaviours.<br>- AI Patrol System, Driving System, Trade System, etc                       |
| Dynamics        | Emergent play behavior that arises when mechanics and systems operate during actual play.<br>- recurring play patterns, pacing, strategic tendencies, etc |
| Player Verbs    | Actions initiated by the player.<br>- hide, craft, hit, etc                                                                                               |
| Game Verbs      | Actions or pressures initiated by the game.<br>- hunt, spawn, time-out, etc                                                                               |
| Resources       | Tracked values or assets that meaningfully affect decision-making, planning or play.<br>- health, ammo, currency, etc                                     |
| Rules           | Explicit or implicit logic that governs states, interactions, constraints, and outcomes in play.<br>- win/lose conditions, unique cases, exceptions, etc  |
| Characteristics | Holistic experiential qualities of the game. (does NOT include raw emotions)<br>- tension, tone, fantasies, etc                                           |
